# Notas y apuntes

Este contenido se **renderiza en la web** desde el archivo Markdown del repositorio:

`content/notas.md`

Puedes usar **negritas**, listas, tablas y enlaces:

| Dónde | Qué |
|-------|-----|
| Servidor | `/opt/calendar-web/repo/content/notas.md` (tras desplegar) |
| Local | la misma ruta en tu clon |

Tras editar el archivo en el servidor, vuelve a construir la imagen Docker para que el cambio entre en la app (el fichero se copia en el build).
